This VM consists of the setup needed to run and test an Eclipse plugin called NavClus(NavMine). The VM has the required versions of JAVA and Eclipse installed and the NavClus plugin has been added to Eclipse.

The details about this project, the source code for the tool, the paper in which the plugin was first described and the plugin itself are available at the following github link. 
https://github.com/SoftwareEngineeringToolDemos/ICSE-2013-NavClus.

If you wish to reinstall the plugin to Eclipse, the steps have been provided in Installation.txt file present on the Desktop. This plugin has been tested with Eclipse Luna version and java 8. If you want to reinstall the entire IDE, please make sure that these requirements are met.
